# IFPlan Leite à Pasto - Privacy Policy

This is the static website for the Terms of Use and Privacy Policy of the **IFPlan Leite à Pasto** application.

## Structure

- `index.html`: Main HTML file with the policy content.
- `styles.css`: Stylesheet for the page.

## Deployment

### Vercel

1. Import this repository into Vercel.
2. Vercel will automatically detect the static files.
3. Deploy.

### Local Development

Open `index.html` in any web browser.
